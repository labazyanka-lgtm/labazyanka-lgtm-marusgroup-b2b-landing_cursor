# Фото объектов

Сюда складываем фото объектов для блока «Опыт» (`components/Experience.tsx`).

## Что положить

- Файлы в формате `.webp` (предпочтительно) или `.jpg`.
- Соотношение сторон 16:10, минимальная ширина 1280 px.
- Имя файла = slug объекта (латиница, kebab-case).
  Например: `eleven.webp`, `city-bay.webp`, `level-prichalniy.webp`.

## Как привязать к карточке

В `components/Experience.tsx` к нужному элементу `PROJECTS` дописать поле:

```ts
image: { src: "/projects/eleven.webp", alt: "ЖК Eleven, фасадное остекление" }
```
